# Configuration

Settings are stored in `~/.qbit/settings.toml` (auto-generated on first run).

Most settings can be configured through the in-app Settings UI.

## Web search

Enable Tavily-powered web search:

```bash
echo "TAVILY_API_KEY=tvly-..." >> .env
```
